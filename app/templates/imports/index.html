Импорт Excel/CSV

Поддерживаются файлы .xlsx и .csv. Основной файл импортируется в сущности системы, а файл договоров можно приложить для автоматического заполнения пустых полей.

Если добавить файл договоров, система попробует обогатить записи по ИНН и адресу.

Поддерживаемые поля

Можно использовать разные заголовки колонок. Для базы договоров уже настроены поля номера договора, даты, адреса объекта, контактного лица, объёма и периодичности вывоза.

Примеры: адрес / address, наименование организации, юридическое наименование, категория объекта по нормативу, ИНН, №, дата, адрес объекта.

Очистить предыдущую загрузку

Удаляет все ранее импортированные объекты из базы данных. После очистки можно загрузить новый основной файл и новую базу договоров.
